Controls
========

Arrow keys or WASD for moving around, mouse for view angle.
Ctrl for up, shift for down.
F1 opens the settings dialog for display mode etc.
Space cycles between showing FPS, Profiler and neither.
Escape for exiting the demo.

Compiling code
==============

To compile this code, you need Framework4.zip from my site.
If there's a problem with compiling the code, make sure that you
got the right framework, and that you have a recent enough copy of it.
Also make sure you have the recent enough version of dependent SDKs.
If it still won't compile, refer to my site for ways to contact me.
